# 宝藏大冒险
## 难度：白银

传说有一个宝藏在某地，小码哥碰巧拿到了关于这个宝藏的地图，为了快点拿到宝藏于是他选择走直线，起点标记为 1，宝藏的位置标记为 n ，路途中凡是坐标为质数的地方都存在着怪物，但这些怪物如果获得了一点食物就会放行小码哥，请帮助小码哥计算出会遇到多少只怪物，以方便他在出发前准备好充足的食物。

#### 格式
>输入格式：
一行一个正整数n表示宝藏的位置<br>
<br>输出格式：
一行一个整数，表示小码哥遇到的怪物数量

#### 样例 1
>输入：
100<br>
输出：
25

#### 备注
>其中：0 < n ≤ 100000

#### 代码
```C++
#include<bits/stdc++.h> 

using namespace std;

int main( )
{
    int a,amount;
    cin>>a;

    for(int i=1;i<=a;i++){
        for(int j=2;j<i;j++){
            if(i%j==0){
                amount++;
                break;
            }
        }
    }
    cout<<a-amount-1;
    return 0;
}
```
